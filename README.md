# 🌤️ CloudBurst Weather Portal

A sleek, responsive frontend weather application utilizing asynchronous JavaScript to fetch real-time meteorological data for cities across the globe.

## 🚀 Features
* **Live Search:** Look up any global city to instantly get current conditions.
* **Real-time Metrics:** Displays temperature in Celsius, weather description, humidity percentage, and wind speed.
* **Zero-Auth API Integration:** Connects seamlessly to a live weather engine without requiring complex server-side authentication keys.
* **Modern Aesthetic:** Styled with a clean, user-friendly dark theme optimized for all devices.

## 🛠️ Tech Stack
* **HTML5** - Structure and semantic layout
* **CSS3** - Custom typography, responsive layout, and dark-theme aesthetics
* **JavaScript (ES6)** - Native Fetch API using `async/await` for asynchronous operations
